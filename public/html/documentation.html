Introduction

Shindow is a web graphic user interface which allows you to take controll of your resources without need of a console.

It is composed by two parts. The first one is a NextJS client which runs in any browser and consumes an API made with Node + Express. This is why you can run Shindow in many different ways.

You can run both the frontend and the backend in the same machine, each one in different machines or even you can consume the API from your own client.

Installing client + backend in the same machine

This is the simplest way to install Shindow. All you have to do is download the application from one of the available sources
