- NEXT_PUBLIC_SECRET: Secret key for encrypting data. Must match backend SECRET.

- NEXT_PUBLIC_KEY_FILE_MAX_SIZE: Max size (in bytes) for key file. Example: 1024 = 1 KB.

- NEXT_PUBLIC_TZ: Timezone for the frontend. Example: "America/Argentina/Buenos_Aires".

- NEXT_PUBLIC_BACK_BASE_URL: Backend base URL. Example: 'http://localhost:5000'.

- NEXT_PUBLIC_API_BASE_URL: API endpoint URL. Example: 'http://localhost:5000/api'.

- NEXT_PUBLIC_INITIAL_PATH: Default path after login. Example: '/'.

- NEXT_PUBLIC_CLIENT_BASE_URL: Frontend base URL. Example: 'http://localhost:3000'.
