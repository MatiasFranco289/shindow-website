The free web graphic user interface

Shindow is a free web graphic user interface for server. It allows you to connect and manage your servers from a web client.

What is Shindow?

Shindow is a web client which allows you to visualize and manage the content of your server. You can run it locally in your own machine, in the same server or even in another machine. It provides features like creating, deleting, uploading and managing your resources efficiently through an intuitive interface with an easy setup.

What can i do with Shindow?

You can login in different ways

You can navigate through directories

You can manage your resources

You can upload one or multiple resources at once
